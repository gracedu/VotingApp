Voting App : Vote for the favorite fruit

This project involves html, php, Vue.js, and css.

Students need to enter their name to login.

Database information: The database is named "vote". It contains two tables "items" and "users".
                      "items" stores the fruit name and the number of vote corresponding to each fruit.
					  The fruits are apple, orange, banana, and pineapple.
					  "users" stores the students' name.
					  There are four students: Jane, John, Mary, and Recker.
Notice: The username and password for the database are not real in this code in order to prevent
        hacking.
